# Problem 6: String List

**Problem link: [here](https://www.practicepython.org/exercise/2014/03/12/06-string-lists.html)**

## Problem Statement

Ask the user for a string and print out whether this string is a palindrome or not. (A palindrome is a string that reads the same forwards and backwards.)

## Example

Output:

```bash
Enter string: Lol
Lol is palindrome.

Enter string: Tesla
Tesla is not palindrome.
```
